## 스크린샷 캡처 가이드

실제 게임에서 캡처할 **핵심 장면 8개** (가로 방향으로 권장):

1. **로비 화면** — 빠른 매칭/커스텀 방/랭킹 진입 버튼 보이게
2. **게임 진행 중** — 4인 플레이어 + 테이블 카드 + 내 손패가 잘 보이는 상태
3. **폭탄 장면** — 폭탄 인터럽트 순간 이펙트
4. **티츄 선언** — 라지 티츄 선언 시 골드 이펙트
5. **용 카드** — 드래곤 카드를 내는 순간 이펙트
6. **라운드 결과** — 점수 정산 화면
7. **상점/커스터마이징** — 다양한 아바타와 카드 뒷면
8. **랭킹/시즌** — 랭킹 리더보드 화면

**캡처 방법**:
- Android: 볼륨다운 + 전원 동시 누르기
- iOS: 전원 + 볼륨업 동시 누르기
- 웹 dev server 에서 F12 → 디바이스 툴바 (Ctrl+Shift+M) → 1080×1920 or 1290×2796 설정 후 캡처 가능
